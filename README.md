# Fontanelle Milano — Project Bootstrap

Struttura iniziale di progetto per una web app sulle fontanelle del Comune di Milano.

> Questo repository contiene **solo lo scheletro architetturale**: nessuna logica applicativa, nessuna query geospaziale, nessuna UI completa.

## Stack previsto
- **FastAPI** per API future e integrazione con MongoDB.
- **Flask** per rendering web server-side e orchestrazione pagine iniziali.
- **HTML + JavaScript** per frontend progressivo.
- **MongoDB** per persistenza dati geospaziali (integrazione futura).
