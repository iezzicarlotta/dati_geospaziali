🎯 RESPONSE FORMATS
═══════════════════════════════════════════════════════════════════════════════

All responses are JSON with consistent structure:

✅ Success Response (200 OK)
   {
     "items": [...],           ← Array of fountains or NILs
     "total_count": 45,        ← Total matching records
     "page": 1,                ← Current page (if paginated)
     "total_pages": 3,         ← Total pages (if paginated)
     ...
   }

✅ Error Response (400/500)
   {
     "detail": "Error message explaining what went wrong"
   }

✅ Fountain Object
   {
     "id": "507f1f77bcf86cd799439011",
     "object_id": "1001",
     "nil_id": 1,
     "coordinate": {"longitude": 9.190, "latitude": 45.464},
     "name": "Fontanella Centro",
     "municipio": "Milano",
     "cap": "20100"
   }

✅ Nearby Fountain (with distance)
   {

     "distance_meters": 245.5
   }

✅ Statistics Entry

📊 EXAMPLE USAGE IN FRONTEND
═══════════════════════════════════════════════════════════════════════════════

Display fountains on map:
  ┌─────────────────────────────────────────────────────────┐
  │ const map = L.map('map').setView([45.464, 9.190], 11);  │
  │ fetch('/api/v1/fountains/search/by-nil', {              │
  │   method: 'POST',                                        │
  │   body: JSON.stringify({nil_id: '1'})                    │
  │ })                                                        │
  │ .then(r => r.json())                                     │
  │ .then(data => {                                          │
  │   data.items.forEach(f => {                              │
  │     L.marker([f.coordinate.latitude,                     │
  │               f.coordinate.longitude])                   │
  │      .bindPopup(f.name)                                  │
  │      .addTo(map);                                        │
  │   });                                                     │
  │ });                                                       │
  └─────────────────────────────────────────────────────────┘

Display statistics table:
  ┌─────────────────────────────────────────────────────────┐
  │ fetch('/api/v1/fountains/statistics/nils')              │
  │ .then(r => r.json())                                     │
  │ .then(data => {                                          │
  │   data.statistics.forEach(stat => {                      │
  │     table.addRow(stat.nil_name,                          │
  │                  stat.fountain_count,                    │
  │                  stat.color_class);                      │
  │   });                                                     │
  │ });                                                       │
  └─────────────────────────────────────────────────────────┘

Display choropleth:
  ┌─────────────────────────────────────────────────────────┐
  │ const colorMap = {                                       │
  │   "LOW": "#2ecc71", "MEDIUM": "#f39c12",                │
  │   "HIGH": "#e74c3c", "VERY_HIGH": "#c0392b"             │
  │ };                                                        │
  │ fetch('/api/v1/fountains/choropleth')                    │
  │ .then(r => r.json())                                     │
  │ .then(data => {                                          │
  │   L.geoJSON(data, {                                      │
  │     style: f => ({                                       │
  │       fillColor: colorMap[f.properties.color_class],     │
  │       fillOpacity: 0.7                                   │
  │     })                                                    │
  │   }).addTo(map);                                         │
  │ });                                                       │
  └─────────────────────────────────────────────────────────┘
